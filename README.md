# PawsForward Animal Charity Website

A modern, compassionate, and visually stunning website for PawsForward, an animal charity organization dedicated to the rescue, rehabilitation, and rehoming of animals in need.

## Features

- **Hero Section**: Engaging hero area with a scroll-triggered zoom effect and a mission statement.
- **Mission Ticker**: A dynamic ticker banner emphasizing the core pillars of the organization.
- **Programs**: Detailed overview of core programs including Rescue, Rehabilitation, Adoption, Community Education, Wildlife Protection, and Veterinary Aid.
- **Impact Stats**: Animated counters showcasing the tangible difference made by the charity.
- **Animals Gallery**: Filterable gallery of animals currently available for adoption or in rehabilitation (Dogs, Cats, Rabbits, Birds, Small Pets, and Wildlife).
- **Testimonials**: Stories from adopters and volunteers using a smooth carousel.
- **Donation Form**: Interactive donation form with preset amounts and secure payment notes.
- **Volunteer CTA**: Encouraging visitors to join as volunteers.
- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices.
- **Rich Aesthetics**: Premium design using custom typography (Playfair Display, Lora, DM Sans) and a warm earthly color palette.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS for styling, grid/flexbox layouts, and custom animations.
- **JavaScript**: Interactive features like the mobile menu, scroll reveals, stats counters, gallery filtering, and testimonial carousel.
- **FontAwesome 6**: High-quality social media and UI icons.
- **Google Fonts**: Premium typography.

## Setup Instructions

1.  Clone the repository to your local machine.
2.  Open `index.html` in any modern web browser.
3.  Ensure you have an active internet connection to load external assets (fonts, icons, and images).
